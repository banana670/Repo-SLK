# Repo-SLK

SLK is a Python script for educational purposes that analyzes URLs
and detects potentially suspicious links.

## Requirements
- Python 3
- requests
- numpy
- scikit-learn

## Install & Run (Termux / Linux)

```bash
git clone https://github.com/banana670/Repo-SLK.git
cd Repo-SLK
pip install requests numpy scikit-learn
python slk.py
